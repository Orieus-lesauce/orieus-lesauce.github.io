    Windows
    Software

    Microsoft-Activation-Scripts - open source - scripts para activar Windows y Office
    WizTree - gratis - analizador de espacio ocupado en tu disco duro
    yt-dlp - open source - línea de comandos para descargar audio/video en miles de sitios
    youtube-dl-gui - open source - fork que añade una interfaz gráfica a yt-dlp
    qBittorrent Enhanced Edition - open source - cliente bittorrent con funciones extra
    OBS Studio - open source - capturador de pantalla para grabar videos y streaming, con soporte de plugins de la comunidad
    Notepad++ - open source - alternativa al editor de texto integrado con funciones útiles
    LocalSend - open source - compartir archivos entre dispositivos en tu red local
    ImHex - open source - hex editor con funciones útiles
    WinUtil - open source - utilidad para debloat, configurar Windows
    Sunshine - open source - host streamer de tu PC a otros dispositivos en tu red local con un cliente Moonlight
    Czkawka - open source - limpiador y organizador de archivos innecesarios de tu PC
    Vencord - open source - cliente de Discord con más funciones
    EarTrumpet - open source - mezclador de audio para Windows que prefiero sobre el nativo de W11
    WinPaletter - open source - personalizador de temas de Windows
    Beyond Compare - prueba gratis - explorer FTP para comparación y sincronización de archivos
    WinSCP - open source - cliente SFTP/FTP para transferir archivos
    Delfinovin - open source - permite usar tu control de Gamecube como Xinput
    Bitwarden - open source - gestor de contraseñas que sincroniza tus claves en tus dispositivos
    Waterfox - open source - fork de Firefox centrado en la privacidad y rendimiento

    uBlock Origin - open source - bloqueador de anuncios y rastreadores
    Privacy Badger - open source - bloqueador de rastreadores en URLs
    Dark Reader - open source - añade modo oscuro a sitios web que no lo tienen
    SponsorBlock - open source - salta automáticamente las secciones de anuncios de YouTube
    Enhancer for YouTube - open source - añade funciones útiles a YouTube
    YT Anti Translate - open source - evita la traducción automática de títulos de videos, audio y descripciones en YouTube
    Tab Session Manager - open source - guarda y restaura sesiones de pestañas
    TwitchAdSolutions - open source - bloqueador de anuncios en Twitch
    Twitch Audio Web - open source - escuchar Twitch en segundo plano para reducir consumo de recursos
    Reddit NSFW Unblocker - open source - desbloquea contenido NSFW sin login en Reddit
